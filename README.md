# Instruments R Us

## Description

As a group, we created an interactive full-stack application.

## User Story

AS a music lover
</br>
I WANT to find and post good, used instruments
</br>
SO THAT I can share my instruments with others

## Installation

- JavaScript
- Express.js
- Handlebars.js
- FileStack API
- MySQL
- Sequelize
- Heroku
- Bootstrap
